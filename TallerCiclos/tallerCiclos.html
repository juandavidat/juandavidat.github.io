Punto 1

Realizar la siguiente figura por medio del ciclo Para

Solución:

          Proceso practicaCiclosPunto1
	
            Definir filas, asteriscos  Como Entero;
            
            para filas<-1 Hasta  10 Con Paso 1 Hacer
              
              Para asteriscos<-1 hasta filas Con Paso  1 Hacer
                
                Escribir Sin Saltar "*";
                
              FinPara
              
              Escribir "";
              
            FinPara
            
          FinProceso

Punto 2

Realizar la siguiente figura por medio del ciclo Mientras - Hacer

Solución:

            Proceso practicaCiclosPunto2
            definir  filas, espacios, asteriscos  Como Entero;
            
            filas<- 10;
              
            Mientras filas >= 1  Hacer
              
              espacios <- 1;
              
              Mientras  espacios <= filas Hacer
                
                Escribir " " Sin Saltar;
                espacios <- espacios + 1;
                
              FinMientras
              
              asteriscos <- 0;
              
              Mientras asteriscos <= (10-filas) Hacer
                
                Escribir "*" sin saltar;
                asteriscos <- asteriscos + 1;
                
              FinMientras
              
              Escribir "";
              filas <- filas - 1;
              
            FinMientras
              
          FinProceso

Punto 4

Se requiere un programa que pueda crear la tabla de multiplicar del número que el usuario indique por medio del ciclo Para; esta debe ser impresa del 1 al 10. Ejemplo, si el usuario indica que desea crear la tabla del 5, entonces el resultado debería ser el siguiente:

Solución:

          Proceso practricaCiclosPunto4
            Definir  numeroMultiplicar, multiplicar, resultado  Como Entero;
            
            Escribir "Por favor ingrese el número que desea multiplicar";
            Leer  numeroMultiplicar;
            
            Limpiar Pantalla;
            
            Para multiplicar<-1  Hasta 10 Con Paso 1 Hacer
              
              resultado <- multiplicar * numeroMultiplicar;
              
              Escribir multiplicar, " X ", numeroMultiplicar, " = ", resultado;
              
            FinPara	
            
          FinProceso

Punto 5

Utilizando el ciclo que usted desee, crear un menú de ejecución infinita hasta que el usuario desee terminar dicho ciclo.

Solución:

          Proceso practricaCiclosPunto5
            Definir nombre, saludo Como caracter;
            Definir  opcion Como entero;
            Definir nombreCaptura Como Logico;
            
            nombreCaptura<- falso;
            
            Repetir
              
              Escribir "Menú de usuario";
              Escribir "1. captura nombre";
              Escribir "2. saludar persona";
              Escribir "3. salir del sistema";
              Leer opcion;
              
              segun opcion Hacer
                1:
                  Escribir "Por favor ingrese su nombre";
                  leer nombre;
                  
                  Limpiar Pantalla;
                  
                  Escribir "Hola, ", nombre, ". ¿como estas?";
                  
                2: 
                  si nombreCaptura = Verdadero Entonces
                    
                    Escribir " ";
                    
                  sino 
                    
                    Escribir "Para saludar debes ingresar tu nombre";
                    
                  FinSi
                  
              FinSegun
            
            Hasta Que opcion = 3;
            
          FinProceso

Punto 6

Se está creando una aplicación que va a estar conectada con un prototipo que permita almacenar contactos telefónicos en el dispositivo. Para ello cada contacto debe contener nombre completo, teléfono y organización. Se requiere que la aplicación permita añadir 3 contactos verificando que el número no esté almacenado, buscar contactos almacenados y eliminar contactos si el usuario lo requiere. Recuerde que el sistema debe terminar cuando el usuario así lo indique.

                Escribir "4. Salir del sistema";

                    FinSi
                    
                    si k = 2 Entonces

                          Escribir "Por favor ingrese el teléfono ", k;

                    FinSi

                          Escribir "Por favor ingrese el teléfono ", k;
                          Leer telefonoC;

                        Escribir "1. Buscar contactos";
                        Escribir "2. Eliminar contactos";
                        Escribir "3. Salir del sistema";
                        leer opcion2;
                        
                        Segun opcion2 Hacer
                          
                          1:
                            Escribir "Nombre: ", nombreA;
                            Escribir "Teléfono: ", telefonoA;
                            Escribir "Organización: ", organizacionA;
                            Escribir "**********************************************";
                            Escribir "Nombre: ", nombreB;
                            Escribir "Teléfono: ", telefonoB;
                            Escribir "Organización: ", organizacionB;
                            Escribir "**********************************************";
                            Escribir "Nombre: ", nombreC;
                            Escribir "Teléfono: ", telefonoC;
                            Escribir "Organización: ", organizacionC;
                            Escribir "**********************************************";
                          2:	
                            nombreA <- "";
                            nombreB <- "";
                            nombreC <- "";
                            telefonoA <- 0;
