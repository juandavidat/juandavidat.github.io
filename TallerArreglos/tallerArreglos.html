Punto 1

Crear un vector de tipo Entero con 5 posiciones, llenarlo con información solicitada al usuario. Después de recoger toda la información, se requiere imprimir el índice de cada posición en el arreglo con su valor de la siguiente manera:

Solución:

          Proceso practicaArreglosPunto1
	
            Definir indice Como Entero;
            Definir numeros2digitos como entero;
            Dimension numeros2digitos[5];
            
            Para indice <- 0 Hasta 4 Con Paso 1 Hacer
              Escribir "Por favor ingrese un número de dos digitos";
               leer numeros2digitos[indice];
             FinPara
             
            Limpiar Pantalla;
            Escribir "[0] = ", numeros2digitos[0];
            Escribir "[1] = ", numeros2digitos[1];
            Escribir "[2] = ", numeros2digitos[2];
            Escribir "[3] = ", numeros2digitos[3];
            Escribir "[4] = ", numeros2digitos[4];
            
          FinProceso

Punto 2

Crear un arreglo de números enteros de 20 posiciones, el cual, debe ser llenado con números aleatorios entre 1 y 100; después de haber llenado dicho arreglo, se debe volver a recorrer utilizando un ciclo diferente al que se usó para llenarse e imprimir los números pares e impares. Ejemplo

Solución:

        Proceso practicaArreglosPunto2
	
          Definir indice, pares, impares como entero;
          Definir numerosN Como Entero;
          Dimension numerosN[20];
        
          para indice <- 0 hasta 19 con paso 1 Hacer
            
            numerosN[indice] <- 1+azar(100);
            
          FinPara
          
          pares <- 0 ;
          indice <- 0;
          
          Escribir "Números pares: " Sin Saltar;
          
          Repetir
            
            si numerosN(indice) mod 2 = 0 Entonces
              
              pares <- numerosN[indice];
              
              Escribir sin saltar pares, ", ";
              
            FinSi
            
            indice <- indice + 1;
            
          Hasta Que indice = 20
          
          indice <- 0;
          impares <- 0;
          
          Escribir "";
          Escribir "Números impares: " Sin Saltar;
          
          Repetir
            
            si numerosN(indice) mod 2 <> 0 Entonces
              
              impares <- numerosN[indice];
              
              Escribir sin saltar, impares, ", " ;
              
            FinSi
            
            indice <- indice + 1;
            
          Hasta Que indice = 20
          
          Escribir "";
          
        FinProceso

Punto 3

Imprimir los números primos del 1 al 1000, el resultado debe ser buscado de forma matemática.

Solución:

      Proceso practicaArreglosPunto3
        Definir c, k, n como entero;
        
        Escribir "Números primos: ";
        
        para n <- 1 hasta 1000 con paso 1 Hacer
          k <- 1;
          c <- 0;
          
          mientras k <= n Hacer
            
            si n mod k = 0 Entonces
              
              c <- c + 1;
              
            FinSi
            k <- k + 1;
            
          FinMientras
          
          si c = 2 Entonces
            
            Escribir n;
            
          FinSi
          
        FinPara
        
      FinProceso
